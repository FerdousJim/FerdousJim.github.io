Blockchain is now one of the most emerging fields in the computer science technology for data storage and transaction on online data among researchers and academics for the recent years. Hence, Blockchain implemnetation on exam result storage database is demandable field to work on. Aim of this thesis is to create a web application to store exam results data using blockchain that can perform to store data by an owner who have the authority to input the data and the other users would only view the data usng their hash id. Here, I have used an Ethereum smart contract which is written by solidity programming language to create a contract that perform or execute the code based on some given conditions. I have used ganache framework for testing and debugging and development purpose of the smart contract. I have used the web3.js library which is the official Ethereum Javascript API. It is used to interact with Ethereum smart contracts and some html and CSS codes for designing purpose. In this thesis I have successfully create a web application. Book Recommender System(Machine Learning)

As I was struggling to search books of my interest based on what i have read so far, I took an initiative to make a machine learning book recommender system website, This is a project on machine learning. I have used three datasets in this project. these are books, users who read books, and ratings by the user. NumPy and pandas are two libraries that I have used. At first, I created a list of 50 popular books according to users' ratings. then I used some collaborative filtering methods to make the website recommend books on user's preference. the steps that I took to make the project are:
1.Deleted duplicated data and missing values from my datasets.
2.merged ratings and books datasets based on ISBN
3.calculated the average rating on each book
4.deleted unuseful columns
5.created a table based on users id's from books and ratings and user's datasets
6.selected top users ids who have at least rated 200 votes
7.counted book rating on the basis of the book title
8.calculated distances using vectorization
9.used similarity matrix
10 .finally sorted the book title lists using similarity scores from each particular book.

Movie Recommender System(Machine Learning)

This is my second project on machine learning focusing on recommendation I have collected two datasets for this project. these are the movies dataset and credits dataset. the libraries that i have used are NumPy, pandas, ast, sklearn, and nltk.
At first, I have done data preprocessing(missing duplicates and missing values from the two datasets). then I merged the datasets. From the merged dataset I have cleared out unwanted columns. After that, I used an algorithm to make a list of keywords from a string list of keywords. then I have made tags concatenating all the columns for every movie. Then I have done text vectorization. the technique I have used for text vectorization is called bag of words. I calculated cosine similarity for every vector with others. I didn't use euclidean distance because it is not good for large dimensional vectors. Then I did reverse sorting similarities in each movie to others.
finally, I fetched out the top 5 similar movies for every movie
